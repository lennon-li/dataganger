# Bundled fonts

These self-hosted web fonts are bundled so the app needs **no external/CDN request**
(see the package's no-network trust guarantee). All three are licensed under the
SIL Open Font License 1.1 (OFL-1.1), which permits redistribution and bundling.

- **Inter** — Copyright (c) The Inter Project Authors (https://github.com/rsms/inter)
- **Instrument Serif** — Copyright (c) The Instrument Serif Project Authors
  (https://fonts.google.com/specimen/Instrument+Serif)
- **JetBrains Mono** — Copyright (c) The JetBrains Mono Project Authors
  (https://github.com/JetBrains/JetBrainsMono)

The full SIL Open Font License 1.1 text is available at
https://openfontlicense.org/open-font-license-official-text/

The `.woff2` files were generated by Google Fonts and are unmodified. `fonts.css`
contains the `@font-face` rules pointing at the local files.
