What DataGangeR prints.

R console
cli + custom print methods

Before the Shiny app, DataGangeR is a package. Every artefact it produces — a profile, a spec, a privacy check, a comparison — has a console receipt: a structured printout that's the package's primary UI. These are the canonical examples; match the typography, colour cues, and section structure when extending.
